General Options

NOTE: each TAB for a process group has a help file with details!

Rainfall, Interception, Infiltration

Switch processes on or off. Each of these options have details in the option tabs. If switched off, the proccess is simply skipped. For instance not infiltration means a completely impermeable surface and 100% runoff and storage.

Overland flow

  • 1D uses a kinematic wave on a predefined continues network that connects all cells to the outlet of the area (ldd.map in PCRaster). The network is user defined and is an input, the DEM is not used. The kinematic wave uses a finite difference iteration. This option gives only runoff and no flood.
  • 1D and 2D: if both options are selected a 1D kinematic wave is used for the runoff and a 2D dynamic wave for the flooding. The flooding ONLY takes place when there are channels defined and they overflow. Flood in this case is overflowing channel water.
  • 2D dynamic flow uses the DEM to determine where water flows. The water is distributed over the cells downstream, using a full dynamic wave for with depth-average velocity. The 'flow' tab gives several numerical options. It uses a semi-explicit finite volume solution with a small adaptive timestep (non-iterative).
    The distinction between runoff and flood is user defined by a flood threshold (in m). All water above this threshold is considered flood (hazardous), below is considered runoff.

NOTE: channel flow is always 1D kinematic wave over a channel network (lddchan.map in PCRaster).

NOTE: a kinematic wave is advised for large scale problems and less than optimal DEMs (e.g. SRTM or ASTER DEMs).

Channels/Rivers and groundwater flow

The channel system can be switched off entirely in which case only overland flow is considered. Channels are 1D networks using a kinematic wave for flow. Groundwater and baseflow can be switched on and off. When there are no channels there is no baseflow

Erosion processes

Switch erosion processes on or off. If switched on, sediment dynamics are simulated for all flows. Default suspended matter is simulated, transport equations can be chosen. Optionally bedload can be simulated. These options can be set for overland flow and channel flow separately.

Infrastructure

Switch on or off the effect of buildings and infrastructure (roads, parking lots) and subsurface drainage from streets.
