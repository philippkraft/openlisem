openLISEM: Quick guideline for code development
===============================================
Download MSYS2.0 and install Qt: https://wiki.qt.io/MSYS2 (except for the static Qt build)

OR if you understand MSYS2.0, basically do:
install msys and open window
pacman -Sy
pacman --needed -S bash pacman pacman-mirrors msys2-runtime
pacman -Su
pacman -S base-devel git mercurial cvs wget p7zip
pacman -S perl ruby python mingw-w64-i686-toolchain mingw-w64-x86_64-toolchain

NOTE: info on pacman options: https://devhints.io/pacman

addtionally install qtcreator and libs for lisem
pacman -S mingw-w64-i686-qt-creator mingw-w64-x86_64-qt-creator
pacman -S mingw-w64-x86_64-openmp mingw-w64-x86_64-qwt mingw-w64-x86_64-gdal mingw-w64-x86_64-clang


Lisem code:
Install git: https://git-scm.com/
Install cmake: https://cmake.org
Register at sourceforge or github

create an empty directory "lisem" for the code, e.g. /prgc/lisem
create an empty directory for the compiled executable (do not make it a subdirectory of lisem), e.g. /prgc/lisem-bin

got into the lisem directory and open a command window.
Type:

git clone ssh://vjetten@git.code.sf.net/p/lisem/lisem .

or

git clone https://github.com/vjetten/openlisem.git .
(do not forget the '.')

make a branch for yourself (e.g. "mylisem)
git branch mylisem
git checkout mylisem

Please do NOT merge your code with the remote repository!

open cmake-gui. The first line should piint to the folder 'lisem' the second line to the folder 'lisem_bin'
press configure, choose "codeblocks - MingW makefiles" 

Open Qtcreator and load the file "CMakeList.txt" in the folder lisem

find Aplicattion.cmake and change:
IF(WIN32)
 SET(QWT_BUILD_DIR "c:\\qt\\msys64\\mingw64")
 SET(GDAL_BUILD_DIR "c:\\qt\\msys64\\mingw64")
ENDIF()

to the path wehre you have instaled msys64

You should be able to compile now...

best of luck!
